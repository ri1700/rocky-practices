Tomcat9 설치
tomcat9 다운로드
# wget https://dlcdn.apache.org/tomcat/tomcat-9/v9.0.89/bin/apache-tomcat-9.0.89.tar.gz
압축 풀기
# tar xvfz apache-tomcat-9.0.89.tar.gz
설치
# mv apache-tomcat-9.0.89 /usr/local/poscodx
# cd /usr/local/poscodx
# ln -s apache-tomcat-9.0.89 tomcat
포트 확인(/usr/local/poscodx/tomcat/conf/server.xml)
...

 <Connector port="8080" protocol="HTTP/1.1"
               connectionTimeout="20000"
               redirectPort="8443" />
...
실행
   # /usr/local/poscodx/tomcat/bin/catalina.sh start
   # ps -ef | grep tomcat
방화벽(firewalld, 8080 포트) 열기

/etc/firewalld/zones/public.xml 수정
/usr/lib/firewalld/services/tomcat.xml 작성
방화벽 서비스 재실행
# systemctl stop firewalld
# systemctl start firewalld
브라우저로 확인

중지 시키기

# /usr/local/poscodx/tomcat/bin/catalina.sh stop
서비스 등록(/usr/lib/systemd/system/tomcat.service)
# systemctl enable tomcat
tomcat 서비스 실행/중지/재실행
# systemctl start tomcat
# systemctl stop tomcat
# systemctl restart tomcat
tomcat manager 설정
/usr/local/poscodx/tomcat/conf/tomcat-users.xml 수정
/usr/local/poscodx/tomcat/webapps/manager/META-INF/context.xml 수정
tomcat 재시작
# systemctl stop tomcat
# ps -ef | grep tomcat
# systemctl start tomcat
